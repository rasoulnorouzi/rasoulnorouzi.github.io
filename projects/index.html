CESSC

causal and non-causal sentence classification dataset and model

CESSC provides a curated dataset and a fine-tuned BERT-based model for binary classification of causal and non-causal sentences within social science texts. The work is connected to the paper (Norouzi et al., 2025).

  • Published article
  • Fine-tuned BERT model on Hugging Face
  • Dataset on Hugging Face
  • Dataset on GitHub
  • Source code

Highlights include 1,000 manually annotated sentences, supplementary machine-labeled sentences, scripts for model fine-tuning and evaluation, and benchmark results.

AI SRT Translator

Gemini-powered subtitle translation while preserving SRT timing

AI SRT Translator is a browser-based application for translating .srt subtitle files into multiple languages using Google Gemini models while preserving subtitle numbering, timestamps, and formatting.

  • Live application
  • GitHub repository

The tool supports configurable translation settings, including model choice, temperature, and delay. It currently supports English, Persian, Spanish, French, Arabic, Korean, German, Italian, Russian, Turkish, Chinese (Simplified), and Dutch.
